HTML Boilerplate

Es la estructura básica de toda página web. En el caso de Visual Studio Code lo hacemos aparecer escribiendo html. Después pinchamos en html:5


Encabezados, de h1 a h6

Los usamos para destacar texto por su tamaño.Por ejemplo

Esto está escrito en h3

Esto esta escrito en h2


Salto de línea con br

El salto de linea se utiliza para cambiar de linea con el comando br


Cursiva

La cursiva la podemos utilizar usando el comando em para poner en cursiva alguna palabra


Negrita

La negrita la podemos utilizar para hacer enfasis en nuestra pagina web, se utiliza con el comando strong


Listas

Hay doformas para hacer listas utilizando ul para las listas no numeradas y ol para las listas numeradas cada elemento existente de la lista debe de ir acompañados del comando li para que sepa que es una informacion de la lista

Imagenes

Para poder poner una imagen en nuestra pagina web necesitamos poner como inicio de comando img despues poner scr para darle una dirrecion ya sea de internet o de nuestro ordenador, despues utilizamos alt para darle una explicacion a la imagen para que las personas ciagas sepan que hay en la imagen o por que la pagina no encuentre la imagen en la dirrecion puesta y se sustituya por el texto que nosotros hemos introducido y por ultimo ponemos widht para darle una anchura o height para darle una altura.
