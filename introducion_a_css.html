Color con CSS

Para usar CSS debemos irnos a la parte llamada head y dentro de esta escribir el comando style, dentro de este css tiene dos reglas el selector y estilo que queremos añadir. Primero selecionamos lo que queremos cambiar de color y luego pones corchetes despues introducimos el comando color si queremos cambiar el color de las letras o background-color para el color de fondo y depues selecionamos el color que queremso.


Selecionar por id

Para poder selecionar por ide primero debemos añadir una tarjeta id a aquello que queramos selecionar por ejemplo en un titulo dentro de la etiqueta h1 introduciriamos otra que se llamaria id="titulo" y despues dentro de style pondriamos #titulo para selecionar esa id.

Selecionar por class

Este caso es muy parecido al de poner una id a un comando se hace exactamente igual pero en vesz de escribir id ponemos class esto nos sirve para no tener que a la hora de ponerlo en el style coja todos aquellos comandos acompañados de una estiqueta class. dentro de style tendremos que poner .elnombrequepensemos y asi selecionara todas las tajetas class con ese nombre
